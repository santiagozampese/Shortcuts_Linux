Configure:

    Para usar o criador de atalhos dê permissão ao arquivo "install.sh"
    e execute-o.

    Ele criará um .desktop do executável do criador.

    Você pode mover o .desktop do criador, mas se quiser mover a pasta principal e exclua o .desktop, mova a pasta ao lugar desejado mantendo a estrutura e crie        outro .desktop com o "install.sh".

Como usar:

    Dê um clique duplo no .desktop do criador, e no terminal que abrir digite o local do executavel (ou o nome se tiver certeza que é o correto), Caminho (ou nome) do icone (se houver) e se o .desktop criado irá ser executado no Terminal.

        ATENÇÃO: Os Atalhos são criados na pasta do usuário.

Configure:

    To use the shortcut creator, give permission to the "install.sh" file

    and run it.

    It will create a .desktop file from the creator's executable.

    You can move the creator's .desktop file, but if you want to move the main folder, delete the .desktop file, move the folder to the desired location while maintaining the structure, and create another .desktop file with "install.sh".

How to use:

    Double-click the creator's .desktop file, and in opened terminal enter the executable location (or name if you are sure it is correct), the path (or name) of the icon (if any), and whether the created .desktop file will be run in the Terminal.


        ATTENTION: Shortcuts are created in the user's folder.
